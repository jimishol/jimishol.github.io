Module src.systems.note_system

Manages a 12-note system, syncing MIDI‐driven note activation to joint visuals.

Supports circular shifts, and both “instant” and “offset” activation modes.

Create a new NoteSystem bound to a scene. Initializes 12 Note objects and their previous‐state tables.

NoteSystem:shift (offset)
Shift all note slots by a given offset. Positive values rotate slots right; negative rotate left. Rebinds each Note to the correct joint object afterward.

Parameters:

  • offset number Number of slots to shift

Returns:

    nil
NoteSystem:update (dt)
Update note and bass states based on MIDI input. In “offset” mode, uses timers to delay deactivation. In “instant” mode, applies state changes immediately.

Parameters:

  • dt number Delta time in seconds since last frame

Returns:

    boolean True if any note or bass state changed
