Module JointLayout

Functions

getJointPositions ()
Returns all joint positions on the torus. Generates 12 points indexed 0 through 11.

Returns:

    table {number, number, number}> Map from joint index to {x, y, z}.
getTriangleCenters ()
Computes centers of four triangles defined by joint indices. Triangles use joints (0,8,4), (1,9,5), (2,10,6), and (3,11,7).

Returns:

    table {number, number, number}> List of triangle center coordinates.
getBillboardYaw (labelPos, cameraPos)
Computes the world-Y yaw angle for a billboard to face the camera.

Parameters:

  • labelPos table Three-element array {x, y, z} position of the label
  • cameraPos table Table with fields x, y, z for the camera position

Returns:

    number Yaw angle in radians around the world-Y axis
