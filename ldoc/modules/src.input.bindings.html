actionForKey (key)
Determine the action constant for a given key press.

This method first checks for shift-modified arrow keys, then falls back to the bindings table defined above.

Parameters:

  • key string The key identifier as received from Love2D callbacks.

Returns:

    action or nil The matched action constant, or nil if no binding exists.

Tables

bindings
Table mapping game actions to lists of keys.

Fields:

  • bindings table<action, string[]> An associative table where keys are action constants and values are lists of key strings that trigger them.
