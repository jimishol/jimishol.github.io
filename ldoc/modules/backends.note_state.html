Module backends.note_state

Active MIDI note state backend (pure TCP version).

Listens on a TCP socket for incoming note updates from backend threads, merges those sources into a unified list of active MIDI notes, and exposes lookup functions for circle‐of‐fourths steps.

Functions

poll () Polls all TCP socket(s) and rebuilds the note state.
getActiveNotes () Returns the current list of active MIDI note numbers.
isNoteActive (stepIndex) Checks if a given circle‐of‐fourths step (1–12) is active.
