Module src.backends.fluidsynth.track_active_notes_thread

Active‐notes tracker thread for the Fluidsynth backend.

Spawns the Fluidsynth process, listens for MIDI note‐on/off events, maintains a table of currently active notes, and writes them to a Lua file (active_notes.lua) for consumption by the main thread.
